# Credit Card Manufacturing Analysis

## Project Overview

This project analyzes the credit card manufacturing process to identify production efficiency, rejection patterns, and quality issues using Python and Power BI.

---

## Objective

- Clean and preprocess manufacturing data
- Analyze production and rejection trends
- Build an interactive Power BI dashboard
- Generate business insights for better decision-making

---

## Tools Used

- Python
- Power BI
- Microsoft Excel

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing using Python
4. Exploratory Data Analysis (EDA)
5. Dashboard Development in Power BI
